# FloppaHAT

This Program Need;
<br>

gcc, python, ruby, git
<br>
<code>
  sudo apt-get install gcc python ruby git -y
</code>


To Install Type:

<code>
  git clone https://github.com/TechnikbladeCode/FloppaHAT/
</code>
<br>
<code>
  cd FlopplaHAT/ && gcc compile -o FloppaHAT
</code>
<br>
<code>
  ./FloppaHAT
 </code>
